## 🚀 Featured Project

<div align="center">

### 🏛️ Vantage — Institutional Intelligence Platform

[![Repo Card](https://github-readme-stats.vercel.app/api/pin/?username=KitTran1307&repo=Institutional-Holdings-Dashboard&theme=transparent&hide_border=true&title_color=00D9FF&text_color=C9D1D9&bg_color=0D1117&icon_color=FF6B35)](https://github.com/KitTran1307/Institutional-Holdings-Dashboard)

</div>

> Open-source SEC institutional holdings intelligence. **Free for everyone, forever.**

**Vantage** transforms SEC EDGAR 13F filings into a clean, interactive intelligence platform — explore portfolios of the world's top hedge funds, track stock ownership over time, and compare quarter-over-quarter shifts in institutional strategy.

| Feature | Details |
|---|---|
| 🏦 **Funds Tracked** | 42 top managers — Berkshire, Bridgewater, Soros, Renaissance & more |
| 📅 **Historical Depth** | 10+ years (40+ quarters) of filing history per manager |
| 📊 **Visualizations** | Stacked area charts, pie charts, Q-o-Q diffs |
| 🔍 **Stock Intelligence** | Find every institution holding a stock & track changes |
| ⚡ **Two Modes** | Live FastAPI backend OR fully static GitHub Pages deploy |
| 🆓 **Zero Cost** | No paywalls · No accounts · No API keys needed |
